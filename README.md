# 英语听力朗读应用

一个纯静态的英语听力练习应用，使用 DeepSeek API 生成英文文本。

## 功能特点

- 🎧 语音朗读（支持美音/英音）
- 🎲 AI 自动生成英文文本
- ⚙️ 直接在前端配置 API Key
- 🎚️ 语速调节
- 🎨 现代化 UI 设计

## 使用说明

1. 点击"配置 API"按钮
2. 输入你的 DeepSeek API Key
3. 选择生成主题
4. 点击"生成文本"按钮
5. 点击"开始朗读"按钮

## 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 导入仓库
3. 部署（无需配置环境变量）

## 本地运行

直接用浏览器打开 `index.html` 文件即可，或使用本地服务器：

```bash
npx serve
```

然后访问 http://localhost:3000
